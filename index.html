Projects

Recipes

This project was the first website I write and design from scratch as testament to my true understanding of the basics of HTML and CSS. It's simplicity, coupled with its creativity, highlight my ability to build unique projects with limited resources.

Landing Page

The Landing Page project was a direct implementation of a flexible webpage design made using HTML and CSS Flexbox. The key takeaway of this project was learning how to build flexible responsive designs that offer a suiting UX.

Rock-Paper-Scissors

The Rock-Paper-Scissors project, part of the Odin Project Fundementals course JS module, was the first webpage I build that depends on programmed logic and user interaction. This project demonstrates several aspects of web design, including individual creativity, the use of external fonts, as well as an uncomplicated UI.

Etch-A-Sketch

The Etch-A-Sketch project revolved around the topic of DOM manipulation, despite including several significant features, including optimized user interaction, reliance on user input, and convenient web design. This project remains an all time favorite of mine due to the immense effort it took and the superb outcome it generated.

Calculator

The Capstone Project of The Odin Project's Foundation's course, the Calculator was one of the more challenging yet rewarding projects I worked on, due to it's complex functionality, sophisticated design, and simplified UI. The Calculator Project combined several aspects of the Foundation's course, further honing my CSS Flexbox skills, and putting my over-the-top programming skills to the test.

Sign-up Form

Although basic, the Sign-up Form project was a clear demonstration of my ability as a full-stack web developer to handle HTML forms, while maintaining a pleasant and appealing web design.
